# Contributing

This repository is published from a private development source. Treat the public repository as a release snapshot unless maintainers explicitly open a contribution workflow.

Before opening changes:

- confirm the change is safe to publish and does not depend on private docs or credentials
- avoid adding real platform identifiers, secrets, or workstation paths
- keep examples synthetic and clearly marked as fake
- document any behavior that requires operator-owned credentials

If maintainers later enable external contributions, they may cherry-pick public changes back into the private source repository instead of developing directly here.
